
# Renters

A School of Cities web project examining housing affordability and rental market trends in Ontario.

Built with SvelteKit. To run locally:

```
npm install
npm run dev
```

## Pages

- **Ontario Rent Control** (`/ontario-rent-control`) - Rental rate increases across Ontario cities before and after the 2018 deregulation of rent control, compared against provincial rent control guidelines. Data from CMHC Rental Market Survey.
